What is CGames

Learning to code is something many people want to do nowadays. However, reading through pages of documentation can be a bit intimidating and boring. CGames is here to help. CGames' goal is to teach first time coding students Python3 with short lessons, fun activities, and a leaderboard to track progress.



What We Teach

The CGames lessons cover basic coding conventions, python's built-in functionality, and much more!


How We Teach It

CGames teaches using short lessons. At the end of each lesson, activities are used to reinforce the lessons in a fun, challenging way.


Why Choose Us

CGames uses the same information that most python tutorials have, so why should a user choose to learn from our service? First, we believe that short lessons are key to make users feel acomplished. Then, paired with fun activities, the user can truly understand what their code does. The Leaderboard allows the users to see how his/her progress compares to other users, adding an extra competitive edge.


How To Get Started

To start lesson one, just click the get started button at the top of the page or below. Enjoy your first CGames lessons!
