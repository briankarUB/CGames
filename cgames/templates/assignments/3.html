Loops

Code can be repeated using a loop. Lines of code can be repeated N times, where N is manually configurable. In practice, it means code will be repeated until a condition is met. This condition is usually (x >=N) but itâ€™s not the only possible condition.

Python has 2 types of loops: for loops and while loops.

{% endcall %} {% call section('for', 'For loops') %}

For loops

We can iterate through a list using a for loop, using the for keyword: {% call utils.code('python') %} >>> items = ['Abby', 'Brenda', 'Cindy', 'Diddy'] >>> for item in items: ... print(item) Abby Brenda Cindy Diddy {% endcall %}

A for loop can be used to repeat a certain block of code N times. In this example, N = 4. {% call utils.code('python') %} >>> for i in range(4): ... print(i) 0 1 2 3 {% endcall %} What's that range(4) call? We'll see below!

{% endcall %} {% call section('range', 'range() function') %}

range() function

The range() function lets us iterate through a range of numbers in an efficient way.

At its most basic, we can iterate a range of non-negative numbers on the half-open range [0, n): {% call utils.code('python') %} >>> n = 4 >>> for i in range(n): # 0 (inclusive) to 4 (exclusive) ... print(i) 0 1 2 3 {% endcall %}

We can choose the start point explicitly, if desired. This will iterate through the half-open range [m, n): {% call utils.code('python') %} >>> m, n = 2, 5 >>> for i in range(m, n): # [2, 5) ... print(i) 2 3 4 {% endcall %}

Finally, we can choose the step size so that, for example, we iterate through the range [m, n) in step sizes of s: {% call utils.code('python') %} >>> m, n, s = 4, 21, 8 >>> for i in range(m, n, s): # [4, 21) in steps of 8 ... print(i) 4 12 20 {% endcall %}

{% endcall %} {% call section('while', 'While loops') %}

While loops

When you want to repeat some code as long as a boolean condition is true, you can use a while loop instead. For example: {% call utils.code('python') %} >>> guess = 0 >>> while guess != 5: ... guess = int(input('Guess the number: ')) ... ... print('That was the right number!') {% endcall %}

{% endcall %} {% call section('nested', 'Nested loops') %}

Nested loops

We can combine loops using a concept called "nesting." Since loops are just code themselves, we can put them inside other loops.

For example, here's how we might iterate through a 3-by-2 2D matrix: {% call utils.code('python') %} >>> for row in range(3): ... for col in range(2): ... # Do something with M[row][col] {% endcall %}

Nested loops can be very useful, but remember that it makes your code more complex and can slow your program down if used without a good reason. Think about whether your problem requires nesting before using it.
