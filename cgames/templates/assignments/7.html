Python is an object oriented programming language, meaning that Classes are used to create objects which have functions and variables. Strings are examples of objects. A string named book, has the function book.replace() and book.lowercase(). {% call utils.code('python') %} book = "Moby Dick" book.lowercase() print(book) {% endcall %}

Classes in Python

Python can create virtual objects that can contian variables and methods. An example: {% call utils.code('python') %} class User: name ='' def __init__(self,name): self.name = name def sayHello(self): print('hello ' + self.name) {% endcall %} Lets go through this example and break down each line

{% call utils.code('python') %} class User: {% endcall %} is the class constructor. Just like the function constructor it starts with class, then the classes assigned name then a colon.

When the class constructor is called, the contents of the class are run just like a function {% call utils.code('python') %} name = "" {% endcall %} sets a variable, name, to the empty string {% call utils.code('python') %} def __init__(self,name): {% endcall %} Here is where we get to the fun stuff. This is the initialization function. This function is the first function to run when a new instance of the class is created. It is like the functions we covered, with two inputs self and name.

We know that name is a string but what is self?

self is Python's way of a class referting to itself, hence the name. {% call utils.code('python') %} self.name = name {% endcall %} It is pretty easy to see in this example, self.name is assigned to the string, name, that is passed in through the function.

If a name is not provided, self.name becomes the empty sting variable we assigned earlier

{% call utils.code('python') %} def sayHello(self): {% endcall %} Is another function header, same as what we have covered. {% call utils.code('python') %} print('hello ' + self.name) {% endcall %} Prints the string 'hello' and inputed name of the user.

And here is how to interact with this classes {% call utils.code('python') %} class User: name = "" def __init__(self, name): self.name = name def sayHello(self): print 'hello' + self.name # create virtual objects james = User("James") david = User("David") eric = User("Eric") # call methods owned by virtual objects james.sayHello() david.sayHello() {% endcall %} The output is: {% call utils.code('python') %} hello James hello David {% endcall %}
