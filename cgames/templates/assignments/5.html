Read File

Reading and writing to a file in python is super helpful and very simple. {% call utils.code('python') %} # Define a filename. filename = "cgames.txt" # Open the file as f. # The function readlines() reads the file. with open(filename) as f: content = f.readlines() # Show the file contents line by line. # We added the comma to print single newlines and not double. # This is because the lines contain the newline character '\n'. for line in content: print(line), {% endcall %} The first part of the code will read the file content. All of the lines read will be stored in the variable content. The second part will iterate over every line in the variable contents.

If you do not want to read the newline characters ‘\n’, you can change the statement f.readlines() to this: {% call utils.code('python') %} content = f.read().splitlines() {% endcall %}

While the codes above work, we should always test if the file we want to open exists. We will test first if the file does not exist, if it does it will read the file else return an error. As in the code below: {% call utils.code('python') %} import os.path # Define a filename. filename = "cgames.txt" if not os.path.isfile(filename): print('File does not exist.') else: # Open the file as f. # The function readlines() reads the file. with open(filename) as f: content = f.read().splitlines() # Show the file contents line by line. # We added the comma to print single newlines and not newlines. # This is because the lines contain the newline character '\n'. for line in content: print(line) {% endcall %}

Write File

The code below creates a new file (or overwrites) with the data. {% call utils.code('python') %} # Filename to write filename = "cgames.txt" # Open the file with writing permission myfile = open(filename, 'w') # Write a line to the file myfile.write('Written with Python\n') # Close the file myfile.close() {% endcall %} The ‘w’ flag makes Python truncate the file if it already exists. That is to say, if the file contents exists it will be replaced.

Append File

If you simply want to add content to the file you can use the ‘a’ parameter. {% call utils.code('python') %} # Filename to write filename = "cgames.txt" # Open the file with writing permission myfile = open(filename, 'a') # Write a line to the file myfile.write('Written with Python\n') # Close the file myfile.close() {% endcall %}

Parameters

Here is a list of the parameters for open()
Flag Action
r Open for reading
w Open for writing (will truncate file)
b binary mode
r+ open file for reading and writing
a+ open file for reading and writing (appends to end)
w+ open file for reading and writing (truncates files)
